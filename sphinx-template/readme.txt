
必要なもの


１．python(2.7)のインストール
２．easy_installのインストール
３．easy_install sphinx


# 以下はblockdiagを使って図を描画するとき必要なものです。
画像ファイルの処理ためにPython Imaging Libraryをインストールします。
http://www.pythonware.com/products/pil/

easy_install blockdiag
easy_install sphinxcontrib-blockdiag


# 以下はPDF出力するとき必要なものです。

Report Lab Toolkitのインストール
  rst2pdfから利用するためのPython用PDF生成ライブラリであるReport Lab Toolkitをインストールします。
  http://www.reportlab.org/oss/rl-toolkit/
Python Imaging Library(PIL)のインストール

easy_install rst2pdf
